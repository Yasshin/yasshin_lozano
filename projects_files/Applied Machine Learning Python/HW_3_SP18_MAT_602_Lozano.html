Problem 1¶

Consider the Pima Indian Diabetes data set from the UCI Machine Learning Repository. The data set contains a number of biological attributes from medical reports. All patients in the dataset are females at least 21 years old of Pima Indian heritage.

The features in the dataset include:

  1. Number of times pregnant
  2. Plasma glucose concentration a 2 hours in an oral glucose tolerance test
  3. Diastolic blood pressure (mm Hg)
  4. Triceps skin fold thickness (mm)
  5. 2-Hour serum insulin (mu U/ml)
  6. Body mass index (weight in kg/(height in m)^2)
  7. Diabetes pedigree function
  8. Age (years)
  9. Class variable (0 or 1)

from sklearn.ensemble import RandomForestClassifier
import matplotlib.pyplot as plt

features = bos.columns[1:]
forest = RandomForestClassifier(n_estimators=10000,
                                random_state=0,
                                n_jobs=-1)

forest.fit(X, y)
# get feature importance
importances = forest.feature_importances_

indices = np.argsort(importances)[::-1]

# print feature importance
for f in range(X.shape[1]):
    print("%2d) %-*s %f" % (f + 1, 30, 
                            features[indices[f]], 
                            importances[indices[f]]))

# plot a feature importance chart    
plt.title('Feature Importances')

        importances[indices],
        color='lightblue', 
        align='center')

We will assign class labels to each of the 506 Boston houses in terms of whether their prices are below the first quartile, in between the first quartile and the median, in between the median and the third quartile or above the third quartile. With such setup, the class labels will correspond to 'low', 'fair', 'high' or 'very high', respectively.

Since boston.target contains the housing prices, we can add these target prices to the bos data frame as follows:

In [72]:
bos['MEDV'] = boston.target

And now we create the classes based on the values of MEDV

In [73]:
bos['Labels'] = np.where(
    bos['MEDV'] < np.percentile(bos['MEDV'].values, 25),'low', 
    np.where(
        (bos['MEDV'] > np.percentile(bos['MEDV'].values, 25)) 
        &(bos['MEDV'] < np.median(bos['MEDV'].values)),'fair',
        np.where((bos['MEDV'] > np.median(bos['MEDV'].values)) 
                 & (bos['MEDV'] < np.percentile(bos['MEDV'].values, 75)),'high',
                 'very high')))
