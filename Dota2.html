Predicting the winner and classifying players for Dota2

Introduction

The goal of the project is to predict the outcome of the match based on the heroes picked and to classify different kinds of players in the game based on win rate versus pick rate.

Data Description

The data set consist of 117 columns with 92.6k rows. For the purpose of this project I removed the columns ClusterID, Game_Mode, and Game_Type. The remaining columns are Team_Won and all 113 heroes at the time this data was gathered. Each row of the dataset is a single game with the following features (in the order in the vector): 1. Team won the game (1 or -1) 2. Cluster ID (related to location) 3. Game mode (eg All Pick) 4. Game type (eg. Ranked) 5 - end: Each element is an indicator for a hero. Value of 1 indicates that a player from team ‘1’ played as that hero and ‘-1’ for the other team. Hero can be selected by only one player each game. This means that each row has five ‘1’ and five ‘-1’ values.

Hero_Win_Rate_Train<-function(herow)
{
    herow=herow+1
    Red_Win<-filter(Htrain,Htrain$Team_Won==-1 & Htrain[,herow]==-1)
    Red_Loss<-filter(Htrain,Htrain$Team_Won==1 & Htrain[,herow]==-1)
    Green_Win<-filter(Htrain,Htrain$Team_Won==1 & Htrain[,herow]==1)
    Green_Loss<-filter(Htrain,Htrain$Team_Won==-1 & Htrain[,herow]==1)
    
    RedW<-sum(Red_Win[,herow]==-1)
    GreenW<-sum(Green_Win[,herow]==1)
    Win<-sum(RedW+GreenW)
    
    RedL<-sum(Red_Loss[,herow]==-1)
    GreenL<-sum(Green_Loss[,herow]==1)
    Loss<-sum(RedL+GreenL)
      
    Total_Games<-sum(Win+Loss)
    
    Winrate <- (Win/Total_Games)*100
    print(Winrate)
  
}
#----------------------------------------------------------------------

The points represent different heroes. The colors are seperated by types of heroes.

  1. Red: Heroes that are played by the pros. These will usually be picked more often.
  2. Yellow: Heroes that are fairly easy to use with a low learning curve.
  3. Blue: Heroes who are not picked very often, but the players who pick them play them very well.
  4. Purple: Heroes that not many do not want to play. These heroes are most likely picked as a joke.
  5. Green: Heroes that are just terrible and should never be played.

Calculating winning team

I am determing the winner based on 5 randomly selected heroes and taking the mean of each side. The team with the higher mean will most likely win the match.

#Selecting 5 Random Heroes to create Red Team and subsetting the Hero name and thier Win rate
